Vì sao Docker quan trọng

Docker giúp đóng gói ứng dụng và môi trường chạy vào container, nhờ đó project dễ chia sẻ cho cả nhóm.

Chỉ cần clone source và chạy docker compose là các thành phần có thể hoạt động gần giống nhau trên nhiều máy khác nhau, kể cả macOS hay Windows.
