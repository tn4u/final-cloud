Tổng quan hệ thống

Final Cloud là mô hình mini cloud gồm nhiều thành phần như Web Frontend, Application Backend, Relational Database, Authentication Server, Object Storage, Internal DNS, Monitoring và API Gateway. Trong đó, Web Frontend đóng vai trò là lớp giao diện đầu tiên mà người dùng nhìn thấy khi truy cập hệ thống.

Chức năng chính

  • Hiển thị trang chủ giao diện web tĩnh
  • Điều hướng đến khu vực blog cá nhân
  • Phối hợp demo cùng backend và gateway
  • Dễ triển khai bằng Nginx trong Docker
